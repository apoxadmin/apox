PLUpload uploader, automatically switched between various technologies (html, flash, silverlight, etc).

Before activating this plugin, please download and unpack the PLUpload sources from their website : http://www.plupload.com/. Warning, this plugin is now configured to use version 2.X , 1.5.X is no more supported.

Place the "js/" folder under the directory of the plugin (plugins/uploader.plupload) and rename it to plupload/ .
