== Description ==
Krystal Shop is an ecommerce WordPress theme powered by Elementor and WooCommerce. Krystal Shop has a unique layout which is both trendy and responsive, ideal for any shopping website. Krystal Shop is easily customizable with the customizer settings and provides easy setup. Krystal Shop is available with 1 click demo import feature. Theme demo here : http://krystalwp.spiraclethemes.com/demo9/
